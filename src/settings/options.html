リボンの設定はJSON形式で記述します。

設定例: 

{
  "items": [
    {
      "domain": "prod.example.com",
      "text": "production",
      "description": "本番環境です",
      "hideOnMouseover": true,
      "ribbonColor": "rgb(0, 136, 255, 0.8)"
    },
    {
      "domain": "staging.example.com",
      "text": "ステージング"
    }
  ]
}

  • domain: リボンを表示するドメイン。
  • text: リボンに表示するテキスト。
  • description: リボンに表示するツールチップ。省略可能です。
  • hideOnMouseover: リボンの上にマウスカーソルを移動するとリボンを隠す場合は true、そうでない場合は false。省略した場合 false。
  • ribbonColor: リボンの色。省略可能です。
